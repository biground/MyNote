## 矩阵
### 矩阵乘法
(M\*N)(N\*P) = (M\*P)  
矩阵乘法结果中的第(i,j)个元素是
矩阵M中第i行的向量与矩阵N中第j列的向量的点积

运算法则：
+ (AB)C = A(BC)
+ A(B + C)= AB + AC
+ (A + B)C = AC + BC
+ 没有交换律
