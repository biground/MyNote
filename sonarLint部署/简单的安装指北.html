sonarLint部署

简单的完全安装指北

  1. 点我下载SonarQube社区版
  2. 解压缩到文件夹
  3. 启动SonarQube Server
# On Windows, excute:
C:\path\to\sonarqube\bin\windows-x86-xx\StartSonar.bat

# On other os, excute:
/path/to/bin/[OS]/sonar.sh console
  1. 在浏览器中打开localhost:9000，以默认管理员身份登录（登录名=admin,密码=admin）
  2. 创建新项目
  3. Display name 和 Projet key 可以用项目名字和它的md5值，比如 王者荣耀/ccb61d96c8f867d4f6c412e95c4f173a，然后点set up
  4. 为项目设置一个token
  5. 选择开发语言和系统，就会自动生成命令
  6. 终端进入开发目录，然后运行8中生成的命令，等待结束后，就能在网站中看到分析结果了
